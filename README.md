## Build Requirements

- Boost (headers)
- protobuf 3.0.0 or later
- C++17 compiler
  - clang 5.0 or later
  - gcc 6.1 or later
- C++17 standard library
  - libc++ 7 or later
  - libstdc++ 6 or later
